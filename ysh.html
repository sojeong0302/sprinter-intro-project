유서현

웹 어플리케이션의 전반적인 흐름을 이해하는 개발자가 되고싶습니다.

인적사항

생년월일: 2002.12.30 (만 23세)
MBTI: ISFP
취미: OTT 시청, 카페 가기

코드잇에 들어온 계기

컴퓨터 관련 전공을 복수 전공하며 자연스럽게 웹 개발에 관심을 갖게 되었습니다. 처음에는 프론트엔드를 공부하며, 작은 변화로도 사용자 경험을 개선할 수 있는 점에 흥미를 느꼈습니다.

이후 우연한 기회로 백엔드를 경험하면서, 프론트엔드와 백엔드가 어떻게 연결되어 동작하는지를 이해하는 것이 얼마나 중요한지 깨달았습니다. 그래서 코드잇 풀스택 과정에 들어오게 되었습니다!

경험해본 것들

HTML, CSS, JavaScript와 React를 사용해 프론트엔드를 구현하고, Vercel을 통해 배포를 진행해보았습니다.

FastAPI를 이용해 API 서버를 구성했으며, MySQL과 AWS RDS를 활용해 데이터를 관리해보았습니다. Redis를 적용해 로딩 속도를 개선해보았습니다.

Docker와 EC2를 사용해 서버 환경을 구성해보았고, GitHub Actions를 통해 CI/CD 흐름도 경험했습니다.
