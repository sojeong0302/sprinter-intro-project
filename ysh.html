유서현

프론트엔드와 백엔드를 공부하며 웹 개발 역량을 넓혀가고 있습니다.

개발을 시작하게 된 계기

대학교에서 컴퓨터 관련 전공을 배우며 자연스럽게 개발에 관심을 갖게 되었습니다. 그중에서도 많은 사용자가 이용하는 웹 개발에 특히 흥미를 느꼈습니다. 처음에는 사용자와 맞닿아있는 UI를 구현하며, 작은 변화로도 사용자 경험을 개선할 수 있는 프론트엔드를 중심으로 공부했습니다.

이후 우연한 기회로 백엔드를 경험하면서, 한 영역만 이해하는 것보다 프론트엔드와 백엔드가 어떻게 연결되어 동작하는지를 이해하는 것이 중요하다는 것을 알게 되었습니다. 그래서 현재는 프론트엔드와 백엔드 모두를 공부하며 웹 애플리케이션의 전체 흐름을 이해하려 노력하고 있습니다.

경험해본 것들

HTML, CSS, JavaScript와 React를 사용해 프론트엔드를 구현하고, Vercel을 통해 배포를 진행해보았습니다.

FastAPI를 이용해 API 서버를 구성했으며, MySQL과 AWS RDS를 활용해 데이터를 관리해보았습니다. Redis를 적용해 로딩 속도를 개선해보았습니다.

Docker와 EC2를 사용해 서버 환경을 구성해보았고, GitHub Actions를 통해 CI/CD 흐름도 경험했습니다.

향후 목표

단순히 기능을 구현하는 데 그치지 않고 사용자 경험과 코드 품질을 함께 고려하는 개발자로 성장하고 싶습니다.
